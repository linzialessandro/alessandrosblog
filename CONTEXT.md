# Context: Website

## Glossary

- **Post**: A single distilled collection of thoughts or discoveries. It consists of a title, summary, content (HTML or Markdown), a publication date, and an optional set of tags.
- **Tag**: A keyword used to categorize a Post. 
- **Archive**: The list of all Posts, which can be filtered by Tags.
- **Filter**: The active set of Tags used to reduce the visible Posts in the Archive.
