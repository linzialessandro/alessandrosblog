## Agent Knowledge Graph

All domain documentation and guidelines for this repository are centralized in the OKF structure. 

See the **[Knowledge Graph](docs/knowledge/index.md)**.

### Issue tracker

Issues and PRDs for this repo live as GitHub issues. External PRs are not treated as a triage surface. See `docs/knowledge/agents/issue-tracker.md`.

### Triage labels

The standard label vocabulary is mapped: needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix. See `docs/knowledge/agents/triage-labels.md`.

### Domain docs

Single-context layout: a single Glossary and docs/adr/ folder at the repository root. See `docs/knowledge/agents/domain.md`.
